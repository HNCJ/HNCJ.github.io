.02 简介

我们是一支特立的群体，也许我们来自不同的地方，未来也将去不同的方向，但我们曾经因相同的专业爱好在相同的地方相遇。如果生活注定让我们分离，我们仍愿意把共同的梦想找一个地方存寄。

程序猿/媛？攻城狮？设计尸？无论别人给我们什么头衔，只有我们自己知道：爱生活，爱编程！

HNCJ 组织

.03 组织宗旨

爱交流，爱分享。HNCJ 作为一个 Github 组织，首先我们注重的是开源。我们总以尊重的态度去学习别人的开源作品，以交流的思想共同进步，以分享的方式获得别人的指点和肯定。

爱生活，爱编程。一行行代码在别人眼里看起来枯燥无味，我们却从中获取快乐和满足。

重缘分，重感情。在我们的专业上，我们从不同的地方出发，在HNCJ相遇共同学习，相遇匆匆即是别离，但我们不会为此而淡了彼此的感情。
